Production Dashboard — Power BI Mapping

A visual-by-visual guide to rebuilding the redesigned dashboard natively in Power BI Desktop. Apply the Concept Medical theme JSON first, then build each visual per the table below. Field names assume a star schema with a Date table and a Machine / Item / Batch dimension.

Setup — do this first

  1. Apply the theme: View → Themes → Browse for themes → select Concept Medical Power BI Theme.json (or the Segoe UI variant).
  2. Page canvas: set page size to 1440 × 1024 (Custom) so the layout matches the design. Page background is pre-set to #EEF1F6 by the theme.
  3. Navy command rail: draw a #122248 rectangle down the left (or across the top); drop in logo-cm-horizontal-white.png.
  4. Section nav: use Buttons + Bookmarks (one bookmark per section) or the Page Navigator. Style buttons with the theme's actionButton fill.
  5. Filters: add Slicers for Unit, Batch, Fiscal Year, Month — set each to Dropdown style.

KPI Cards (×6)Card / New card
  • Use the New Card (preview) visual — supports reference labels & the colored accent bar. Or stack: rectangle + Card visual.
  • Values: Units Produced, Plan Attainment %, Capacity Util %, Yield FPY %, Rejection Rate %, Running Batches.

  • Left accent bar: thin rectangle behind each card — color by status (blue / green / amber / red).

03 · Manufacturing Performance

VisualBuild
Batch Progress Overview (Gantt)Custom visual
  • Not a stock visual. Best: "as Timeline / Gantt" or xViz / Craydec Gantt from AppSource. Fields: Task = Batch, Start/End = phase timestamps, Legend = Status.
  • No-custom-visual fallback: a 100% stacked horizontal bar per batch with transparent "offset" segments + colored phase segments. Colors: Scheduled #C9D0DC, Running #1D4E9F, Paused #C77700, Completed #1E9F6E, Failed #B11F29.
Machine Load DistributionClustered bar
  • Horizontal bar chart: axis Machine ID, value Load %, sorted descending.
  • Heat coloring: Format → Bars → Color → fx Conditional formatting → Rules: ≥23% → red #B11F29, 18–23% → amber #C77700, <18% → green #1E9F6E. (Or "Gradient" using theme min/center/max.)
FG Output by ItemClustered bar
  • Horizontal bar: axis Item, value Output, sort descending. Single blue series #1D4E9F, data labels on.
Line Efficiency statsCards / Multi-row card
  • Multi-row card for the four mini-stats, or four small Cards. Bottleneck/Spare callouts = text boxes with badge shapes.

Fidelity expectations. Native Power BI will reach roughly 90% of this design. Rounded corners, soft shadows, and the 14px card radius are handled by the theme. The Gantt and the donut center-labels need the workarounds noted above. For pixel-exact reproduction of any panel, the Deneb (Vega-Lite) or HTML Content custom visuals can render the HTML/SVG directly — useful for the Gantt and bespoke callouts.
